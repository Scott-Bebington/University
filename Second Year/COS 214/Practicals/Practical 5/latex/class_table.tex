\doxysubsection{Detailed Description}
Class that represents a table in the restaurant. 

Each \doxylink{class_table}{Table} represent a node in a linked list. There are previous and next pointers to navigate through the list. Each table has a waiter assigned to it and a pointer to another table if it is combined. There are also other attributes for example the table number, size and its availability. \begin{DoxyAuthor}{Author}
Alcodeholics Anonymous 
\end{DoxyAuthor}
